# 🍳 나만의 레시피 북 — "한 번 장보기로 세 끼"

레시피 에이전트와 함께 만든 자취생 레시피 북입니다.
**정통 레시피는 그대로 유지하되, 세 요리가 재료를 최대한 나눠 쓰도록** 구성해
한 번 장을 보면 재료를 남김없이 소진할 수 있게 했습니다.

## 레시피 목록
1. [마늘 오일 파스타 (알리오 올리오)](파스타.md)
2. [계란 볶음밥](볶음밥.md)
3. [돼지고기 김치찌개](김치찌개.md)

> 보너스: 옆 폴더의 [간장계란밥](../recipe/간장계란밥.md)도 계란·진간장·참기름·식용유·깨소금을 함께 씁니다.

## 공통 재료 한눈에 보기

| 재료 | 파스타 | 볶음밥 | 김치찌개 |
|------|:------:|:------:|:--------:|
| 양파 | ✅ | ✅ | ✅ |
| 마늘 | ✅ | ✅ | ✅ |
| 식용유 | ✅ | ✅ | ✅ |
| 소금 | ✅ | ✅ | ✅ |
| 후추 | ✅ | ✅ | ✅ |
| 진간장 | – | ✅ | ✅ |
| 대파 | △* | ✅ | ✅ |
| 계란 | – | ✅ | – |
| 참기름 | – | ✅ | – |
| 깨소금 | – | ✅ | – |

\* 파스타 마무리 허브를 파슬리 대신 **쪽파**로 쓰면 대파(파)류까지 세 요리가 공유합니다.

## 장보기 리스트 (자취생용)
- **세 요리 공용 (한 번 사면 끝):** 양파 1개, 마늘 한 줌, 식용유 한 병, 소금, 후추
- **두 요리 공용:** 진간장, 대파 (볶음밥 + 김치찌개)
- **개별 구매:** 스파게티 면(파스타) / 계란·참기름·깨소금(볶음밥) / 김치·돼지고기·두부·고춧가루(김치찌개)

## 만든 방법
- 사용 에이전트: **레시피 에이전트** (`.claude/agents/recipe-agent.md`)
- 정통 레시피를 요청한 뒤, "재료를 최대한 공유해 효율적으로 소비"라는 나만의 방향으로 다듬었습니다.
