## 고블린 퀘스트 (강의 미션 시스템)

"고블린"은 하버스쿨의 게임화된 퀘스트 시스템이다. 사용자가 "고블린"이라고 하면
아래 규칙대로 미션 수행을 돕는다.

### 개념
- **흐름**: 기초개념 학습 → **고블린**(배운 걸 바로 써서 직접 만들기 → 리뷰 → 설명 듣기)
  → **퀘스트**(실제 앱 인터랙션을 직접 구현)
- **포인트**: 고블린/퀘스트를 수행하면 포인트 지급, 난이도별 차등.
  하버스쿨 프로필에서 현재 포인트 확인.
- **BONUS-QUEST**: 제한시간 내 빠르게 결과물 만드는 연습 — 단시간 고득점.

### 작업 패턴
- 폴더 구조: `week_N/quest/<순번>_<미션이름>/` (예: `week_2/quest/01_age-calculator/`)
- 보통 단일 `index.html` (CDN React 18 + Tailwind) 한 파일로 구현.
- 에이전트는 자동 호출하지 않는다. 필요하면 `@single-react-dev`로 명시 호출.
  (사용자가 정한 기존 방침 유지)
