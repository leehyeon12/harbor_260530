# 퀘스트 7 — [Context] 나를 설명하는 .md + Q&A 에이전트

나에 대한 정보를 담은 `about-me.md` 컨텍스트를 만들고, **그 파일에 있는 내용만 근거로**
답하는 Q&A 에이전트(서브에이전트)를 만드는 퀘스트입니다.

> 이 과제는 **7·8·9 시리즈**의 첫 번째입니다. 핵심은 동일 — AI를 범용 일반지식이 아니라
> '내 컨텍스트(.md)'를 근거로만 답하게 만들어 AI를 내 방향에 맞추는 것.
> (8: 수업 자료 .md + 메모리 / 9: 서버 + AI 풀스택 앱으로 확장)

## 핵심 아이디어
AI의 일반지식이 아니라 **'내 .md'를 근거로만** 답하게 하는 것이 컨텍스트의 핵심입니다.
파일에 없는 내용은 솔직하게 "모른다"고 답하도록 설계했습니다.

## 파일 구성
| 파일 | 설명 |
|---|---|
| `about-me.md` | LEE의 이력·경력·가치관을 정리한 컨텍스트 파일 (개인정보 특정 항목은 익명화) |
| `agent-prompt.md` | `/agents`에 붙여넣을 설명 / 시스템 프롬프트 문구 |
| `README.md` | 이 문서 |

생성된 에이전트는 프로젝트 루트의 `.claude/agents/` 에 저장됩니다.

## 에이전트 만드는 법
1. Claude Code 입력창에 `/agents` 입력
2. `Create new agent` 선택
3. 저장 위치 → `Project` (이 저장소에 포함되어 제출물로 함께 제출됨)
4. 생성 방식 → `Generate with Claude`
5. `agent-prompt.md`의 **1) 설명**을 붙여넣어 생성
6. 필요하면 **2) 시스템 프롬프트 완성본**으로 다듬기, 도구는 `Read`만
7. `Save`

## 데모 시나리오 (스크린샷용)
에이전트와 최소 2회 이상 대화하며 동작을 확인합니다.

- **있는 내용 질문** → 자료 근거로 답함
  - "이 사람 무슨 일 해?" → 금융권 SAP FI 개발자
  - "대표 성과가 뭐야?" → 투자현황 조회 1시간 → 5분 단축 등
  - "일할 때 중요하게 생각하는 건?" → 사용자가 직접 제어하는 프로그램 지향
- **없는 내용 질문** → "제 자료에는 없는 내용이라 답변드리기 어렵습니다"
  - "취미가 뭐야?", "좋아하는 음식은?" → 자료에 없으므로 모른다고 답함

## 봇 캐릭터
- **전문 비서 톤**: 정중하고 간결하게 핵심만 전달
- 모르는 질문은 지어내지 않고 솔직하게 "자료에 없다"고 답함

## 제출물
- GitHub 저장소 링크 (이 폴더 + `.claude/agents/` 의 에이전트 파일)
- 동작 스크린샷 (최소 1장)
- 에이전트와의 대화 스크린샷 (최소 1장, 필수)

## 확장 아이디어
`about-me.md`만 바꾸면 '친구 소개 봇', '우리 가게 소개 봇' 등으로 그대로 확장할 수 있습니다.
