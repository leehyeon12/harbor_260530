# 퀘스트 8 — [Context+Memory] 수업 내용 Q&A 에이전트

수업에서 배운 내용을 질문하면, **수업 자료(텍스트)와 실습 코드(코드)를 모두 근거로** 답하는
Q&A 에이전트를 만드는 퀘스트입니다. 메모리를 활용해 **이전 대화 맥락도 기억**합니다.

> **7·8·9 시리즈**의 두 번째입니다. 7번이 'about-me.md 1개'를 근거로 했다면,
> 8번은 근거를 **수업 자료 + 실습 코드 전체**로 넓히고 **메모리**(이전 대화 기억)를 더했습니다.

## 핵심 아이디어
- **텍스트 컨텍스트** + **코드 컨텍스트**를 한 에이전트가 함께 참조한다.
- 개념 질문은 노트로, "그때 만든 코드 보여줘"는 실습 파일을 직접 찾아서 답한다.
- 자료에 없으면 지어내지 않고 "모른다"고 답한다.
- **메모리**로 "아까 물어본 거랑 연결해서 설명해줘"가 동작한다.

## 파일 구성
| 파일 | 설명 |
|---|---|
| `class-notes.md` | W1~W3 수업 핵심을 정리한 **텍스트 컨텍스트** (어디를 보면 되는지 가이드 포함) |
| `agent-prompt.md` | `/agents`에 붙여넣을 설명 / 시스템 프롬프트 문구 |
| `README.md` | 이 문서 |

**코드 컨텍스트**는 별도 파일이 아니라 이 저장소의 `week_1`~`week_3` 실습 코드
(`index.html`, `server.js` 등) 그 자체입니다.

생성된 에이전트는 프로젝트 루트의 `.claude/agents/` 에 저장됩니다.

## 에이전트 만드는 법
1. Claude Code 입력창에 `/agents` 입력
2. `Create new agent` 선택
3. 저장 위치 → `Project` (이 저장소에 포함되어 제출물로 함께 제출됨)
4. 생성 방식 → `Generate with Claude`
5. `agent-prompt.md`의 **① 한 줄 설명**을 붙여넣어 생성
6. 필요하면 **② 완성 시스템 프롬프트**로 다듬기, 도구는 `Read`, `Grep`, `Glob`
7. `Save`

## 데모 시나리오 (스크린샷용 — 최소 3장)
에이전트와 2회 이상 대화하며 텍스트·코드·메모리 동작을 모두 보여줍니다.

- **① 텍스트 질문** → 수업 노트 근거로 답함
  - "지난 수업에서 배운 API 라우팅이 뭐였지?" → webserver-02의 `req.url` 분기 / JSON 응답 설명
- **② 코드 질문** → 실습 파일을 찾아 실제 코드로 답함
  - "그때 만든 정적 파일 서버 코드 보여줘" → `week_3/practice/01_webserver-static/server.js` 발췌
- **③ 메모리 활용** → 이전 대화를 이어서 답함
  - "방금 그 서버에서 비밀번호는 어떻게 다뤘어?" → 앞 맥락 + webserver-03 보안 포인트 연결
- (보너스) **없는 내용 질문** → "수업 자료에는 없는 내용이라 답변드리기 어렵습니다"

## 메모리에 대하여
서브에이전트는 호출마다 독립적이라, "이전 대화 기억"은 **같은 세션 내 연속 대화**와
Claude Code의 **메모리 기능**으로 구현합니다. 핵심은 거창한 구현이 아니라,
이미 있는 컨텍스트·메모리 기능을 조합해 *"아까 그거 기억해?"* 가 되게 하는 것입니다.

## 봇 캐릭터
- **수업 조교 톤**: 정중하고 간결하게 핵심만, 코드 인용은 필요한 부분만 발췌
- 모르는 건 지어내지 않고 솔직하게 "자료에 없다"고 답함
